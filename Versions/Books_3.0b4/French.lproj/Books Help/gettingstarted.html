Erste Schritte

Neue Books-Nutzer

Der Umgang mit Books ist denkbar einfach. Wenn Sie diese Hilfedatei lesen können, dann haben Sie Books bereits installiert. Herzlichen Glückwunsch!

Bei dem ersten Start der Anwendung, sehen Sie im linken Fenster ein Element, daß sich «Meine Bücher» nennt. Das ist Ihre erste Bücherliste. Sie können auch ein Fenster sehen, daß «Neues Buch» genannt wird, wenn Sie auf das entsprechende Piktogramm in der oberen Leiste klicken. Die Details der Bücher erscheinen als formatierter Text.

Die standardmäßig angebotenen Informationen sind nicht gerade interessant. Also fügen wir lieber einige Informationen hinzu. Klicken Sie dazu in dem Auswahlfenster auf den Reiter «Infos», der sich oben rechts befindet.Dieses Fenster können Sie auch mit der Tastenkombination Apfel-I, dem Piktogramm «Infos einblenden» oder auch durch das Doppelklicken auf einen Titel aufrufen.

Das Fenster enthält die Metadaten über den entsprechenden Eintrag. Darin ist standardmäßig der Eintrag «New Book» enthalten. Sie können den Titel ändern, indem Sie in das Feld hinter «Titel» klicken und einen neuen Titel angeben. Außerdem können Sie weitere Informationen hinzufügen, indem Sie in das entsprechende Feld die dazugehörigen Angaben eintragen. Haben Sie auch noch das Titelbild des Buches als Grafikdatei können Sie dieses einfach auf das Feld «Titelbild» ziehen und es somit an dieser Stelle einfügen.

 

Wenn Sie auf den Reiter «Weitere Infos» klicken, erscheinen weitere Felder. Die oberen Felder sind weiteren Standardinformationen gewidmet. Die unteren Felder sind für nutzerdefinierte Informationen reserviert. Sie können weitere Felder erstellen und Ihre eigenen Werte eintragen. So können Sie die Informationen der Einträge auf Ihre Bedürfnisse anpassen.

Klicken Sie auf den Reiter «Exemplare», so sehen Sie eine andere Tabelle und damit verbundene Felder. Damit können Sie verschiedene Exemplare eines Buches verwalten. Sie können der Anwendung also mitteilen, wenn sich ein Exemplar in Ihrem Wohnzimmer und einanderes in Ihrem Schlafzimmer befindet. Haben Sie lediglich ein Exemplar des Buches, müssen Sie diese Felder nicht ausfüllen.

Der Reiter «Rezensionen» enthält die Felder, um Ihre Reaktionen und Gedanken zu einem bestimmten Buch festzuhalten. Sie können ebenfalls ihren Lesefortschritt festhalten und das Buch, nachdem Sie dieses zu Ende gelesen haben, bewerten. Auch wenn Sie das Buch mehrere Male lesen, können Sie für jeden einzelnen Lesevorgang eine Rezension festhalten.

Schließlich gibt es da noch den Reiter «Ausleihgeschichte». Diese Angaben helfen Ihnen, den Überblick zu bewahren, an wen Sie wann welches Buch entliehen haben und wann Sie sie zurückbekommen sollten. Das richtige Werkzeug für denjenigen, der Bücher sucht und nicht mehr weiß, ob er diese an irgendjemanden ausgeliehen hat.

Nutzer einer vorherigen Version von Books

Haben Sie bereits Books 2 genutzt, ist es ganz einfach, Ihre Daten für Books 3 aufzubereiten. Starten Sie dazu die Anwendung und wählen Sie in dem Menüpunkt «Bearbeiten» den Eintrag «Importieren» und dort den Eintrag «Books 2.0». Sie erhalten ein Fenster, durch das Sie ihre .books-Datei auswählen können. Nachdem die Erweiterung die Datei bearbeitet hat, erhalten Sie eine Liste aller Bücher, aus denen Sie alle Bücher auswählen können, die Sie mit Books 3 verwenden wollen. Wollen Sie alle Ihre Bücher importieren, dann klicken Sie einfach auf den Knopf «Importieren». Das Fenster verschwindet und Books gibt die Meldung aus, daß Ihre Dateien importiert werden. Sobald der Vorgang abgeschlossen ist, erhalten Sie eine neue Bücherliste mit dem Namen «Daten von Books 2.0», die Sie auch nach Ihren Wünschen umbenennen können.
