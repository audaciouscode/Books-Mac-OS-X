Daten importieren & exportieren

Auch wenn Sie Books zur Katalogisierung und Verwaltung Ihrer Sammlung nutzen, gibt es auch noch andere Möglichkeiten, die für Sie interessant sein können. Wenn Sie sich mit XML-Dateien auskennen und diese verwenden wollen, so können Sie mit dem «Books Folder Exporter» die gesamte Bibliothek in einen Ordner exportieren, der die Bilder, Informationen und die XML-Datei Ihrer Sammlung enthält. Möchten Sie die Informationen in einer Datenbankdatei verwenden, steht Ihnen der «Tab-Delimited Text Exporter» zur Verfügung. Möchten Sie sichergehen, daß Sie ein Buch nicht zweimal kaufen, so können Sie Ihre Sammlung auf den iPod laden, nachdem Sie diese mit dem «iPod Exporter» exportiert haben. Für Mobilfunktelephone steht der «Books Micro Edition Exporter» zur Verfügung.

Ihre Daten zu exportieren, ist mit Books nicht sehr schwer. Wählen Sie einfach die Bücherliste aus, die Sie exportieren möchten. Unter dem Menüpunkt «Bearbeiten» wählen Sie dann den Punkt «Exportieren» und dort den Eintrag aus, der Ihren Bedürfnissen entspricht. Eine Erweiterung erscheint und fragt nach zusätzlichen Informationen (zum Beispiel danach, wohin Ihre Daten gespeichert werden sollen). Nachdem Sie alle Fragen beantwortet haben, wird die Bücherliste exportiert und die Erweiterung gegebenenfalls geschlossen.

Auch das Importieren von Daten ist wirklich einfach. Books kann Daten von Dateien, aus Datenbanken und von Servern lesen. Sie werden wahrscheinlich gefragt, wo sich die Datei befindet, die Sie importieren wollen. Einige Erweiterungen erfragen auch andere Informationen. Nachdem die Dateien importiert worden sind, erscheint in Books eine neue Bücherliste mit allen Informationen. Sie können die Bücher in andere Listen verschieben oder auch die neue Bücherliste wieder löschen. Um eine Datei zu importieren, wählen Sie in dem Menüpunkt «Bearbeiten» den Punkt «Importieren» und die gewünschte Erweiterung.

Kennen Sie eine Möglichkeit, Daten zu importieren oder zu exportieren, die in Books eingefügt werden soll, sehen Sie sich bitte die nächste Seite an..
