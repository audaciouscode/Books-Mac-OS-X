Customizing the Main Display

In the lower right portion of the main window is the book display. This is used show the relevant details about the selected books.


The Default Display

The look and feel is customizable by selecting a display plugin from the preferences. Books ships with several styles, so find the one that suits you best.

In addition to the look and feel, you can customize what fields are visible, including custom fields. To define what fields you would like to see, open the preferences and click the Details tab.


A Customized Display Using the Books Online Plugin

The example above shows the preference configuration used to show two standard fields and one custom field. Note that Display default fields... box has been unchecked to allow the customization.

Did You Know?

Books display plugins use the same rendering technology as Apple's Safari web browser. If you are comfortable with HTML and CSS and would like to further customize your display, contact us and we'll get you all of the things you need to get started.
