Getting the Data on Your Palm Device

Now that you've exported your data, there are a few more steps before the data is usable on your device.

1. If you haven't already, download Pilot-DB and follow the instructions to install it on your device.

2. Locate the file exported by Books. It will end in a ".pdb" extension.

If you use The Missing Sync for PalmOS:

3. Launch the application. Drag the exported ".pdb" file to the internal memory window. (Pilot-DB does not seem to be able to find databases installed on memory cards.)

If you use Palm's Send to Handheld application:

3. Drag the exported ".pdb" file to the Send icon.

4. Sync your Mac and Palm device. The files will be installed to your Palm and can be accessed by starting the "DB" application. For notes on using the Pilot-DB application, see the Pilot-DB manual.
